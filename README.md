# Portfolio BTS SIO

Portfolio statique en HTML, CSS et JavaScript, pensé pour un rendu professionnel futuriste techno et une publication simple sur GitHub Pages.

## Personnalisation rapide

1. Remplace `ton-pseudo` dans `script.js` par ton nom d'utilisateur GitHub.
2. Modifie les textes dans `index.html`: option SLAM ou SISR, projets, stage, alternance, email et LinkedIn.
3. Ajoute tes vrais dépôts GitHub dans la section `Projets`.
4. Garde une trace de tes preuves BTS SIO: captures, liens, documentations, commits, fiches de veille.

## Publication GitHub Pages

1. Crée un dépôt GitHub, par exemple `portfolio-bts-sio`.
2. Envoie ces fichiers dans le dépôt.
3. Dans GitHub: `Settings` > `Pages` > `Deploy from a branch`.
4. Choisis la branche `main` et le dossier `/root`.
5. Le portfolio sera disponible à l'adresse GitHub Pages indiquée par GitHub.

## Fichiers

- `index.html`: contenu du portfolio.
- `styles.css`: style futuriste, responsive et animations.
- `script.js`: menu mobile, filtres projets, animations et lien GitHub.
- `assets/hero-tech.png`: image d'ambiance utilisée dans l'accueil.
